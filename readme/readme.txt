Plugin for CudaText.
It gives intel-commands for JavaScript lexer, using Tern engine, http://ternjs.net/

- Auto-completion (Ctrl+Space)
- Goto-definition (item in editor context menu, finds declaration of name)
- Show function-hint (Ctrl+Shift+Space, shows function's arguments hint)

Lexer name "JavaScript" is set in the file install.inf, you may append names of other JS-based lexers, after comma.
